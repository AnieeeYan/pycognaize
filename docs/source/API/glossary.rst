Glossary
========

.. glossary::

    AI Interface
        Definition of input and output fields.

    snapshot
        Snapshot is an immutable collection of annotated documents,
        which contains the following:

        * input and output fields of a predefined :term:`AI Interface`
        * Original PDF document
        * OCR data of all pages of the document
        * Images of all pages of the document

    OCR
        Optical Character Recognition
    Cell
        Cell is a single cell of a table in TableTag.
